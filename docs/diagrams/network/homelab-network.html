Ravil's Homelab Network Architecture

Point-in-time topology: UniFi edge, Proxmox, Talos Kubernetes, Komodo/Docker, ingress, identity, storage, and observability.

Snapshot: 2026-06-14
Edge UniFi Express WAN/LAN and Proxmox Traefik LXC route traffic into K8s or Komodo by SNI.
Kubernetes Talos nodes run Cilium, Gateway API, ArgoCD, cert-manager, Sealed Secrets, and internal/external app routes.
Docker Komodo manages Compose stacks; Docker Traefik terminates TLS for catch-all hostnames.
Data/Auth PocketID/OIDC protects apps; Proxmox, PBS, SMB, and Docker volumes hold persistent state.
